Below you see an image of the color selector. With this color selector you can select a default color, a color from a fixed set and you can construct a color using the standard system color selection dialog.

The color selector is devided into three sections:

Default Color When you click on this button the default color for the specific selection is selected. The resulting color varies between the different color entries you can edit.
Fixed color button With the 40 fixed color buttons you can select a predefined color.
Custom Color... Clicking on this button will open the standard system color selection dialog. You can use this dialog to create a color specific to your needs.
