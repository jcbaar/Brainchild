This is the key combination recorder dialog. With it you can edit record a new key combination to map.

Edit control This is a special edit control. It allows you to type any combination of keys including qualifier keys like Shift, Control and Alt. When, for example, you type the control key plus the F1 key the control will show:

Ctrl F1

When you have typed a mappable key combination the OK button is enabled which allows you to select the typed combination.
OK Clicking on this button confirms the changes made.
Cancel Clciking on this button cancels the changes made.
