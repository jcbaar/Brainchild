BCM_SETUNICODEFLAG

BCM_SETUNICODEFLAG
	wParam = ( BOOL )bUnicode;
	lParam = 0;

Set's or clears the Unicode save flag.

  • Returns the previous state of the Unicode flag.
bUnicode
When this is non-zero the file will be saved as a Unicode text file the next time BCM_SAVEFILE is invoked. When this is zero the file is saved as a normal text file.
