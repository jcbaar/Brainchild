BCM_SETCOLOR

BCM_SETCOLOR
	wParam = ( WPARAM )( int )nColorIndex; 
	lParam = ( LPARAM )( COLORREF )crColor;

Changes the RGB value of the specified color.

  • Returns the previous COLORREF value of the specified color or CLR_NONE uppon an error.
nColorIndex 
The index of the color to change. This must be any of the following values.

  • BCC_TEXT
  • BCC_BACKGROUND
  • BCC_BACKGROUND_READONLY
  • BCC_SELECTION_MARGIN
  • BCC_BOOKMARK
  • BCC_NUMBER
  • BCC_DELIMITER
  • BCC_SELECTED_TEXT
  • BCC_SELECTED_BKGND
  • BCC_LINE_NUMBERS
  • BCC_BACKGROUND_LINE_NUMBERS
  • BCC_BACKGROUND_NUMBER
  • BCC_BACKGROUND_DELIMITER
  • BCC_BRACKET_MATCH
  • BCC_LINE_HIGHLIGHT
  • BCC_HYPERLINK
  • BCC_BACKGROUND_HYPERLINK
  • BCC_MARKER_1
  • BCC_MARKER_2
  • BCC_MARKER_3
  • BCC_MARKER_4
  • BCC_MARKER_5
  • BCC_MARKER_FG_1
  • BCC_MARKER_FG_2
  • BCC_MARKER_FG_3
  • BCC_MARKER_FG_4
  • BCC_MARKER_FG_5

A value of CLR_DEFAULT will set the default color for the selected color index.
crColor
The COLORREF value to set in the specific color index.
Note:
With this function you change the specific color for all Brainchild controls which have been created by the calling thread.
