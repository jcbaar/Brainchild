This is the properties dialog. It displays information about the text currently in the control. You can also use it to change the file mode and tab size settings. In addition to the file information it also displays the registered file type icon if one exists. If not a default icon is displayed. In the dialog as displayed above you have the following possibilities:

Filename

This displays the fill path of the text currently in the control.

File Type

When the file type of the text curently in the control is registered this will display the type description of the file. If the file type is not registered this will show <Unknown>.

Text Class When the text in the control is of a text class supported by the Brainchild control this will display the name of the text class as edited on the General page of the brainchild settings program.
Mode This displays the current file mode. Either MS DOS, Unix or Macintosh. By selecting another file mode you change the setting for the active text class of all brainchild controls created by the program.
Size This displays the file size in characters and number of lines.
Unicode Tells wether or not the file is a Unicode text file.
Last Saved This displays the time and date at which the file was saved last.
Tab Size Displays the size of one tab-jump of the active text class. Changing this value will change the setting for the active text class of all Brainchild controls created by the program.
OK Clicking on this button will activate the changed file mode and/or tab size setting.
Cancel Clicking of this button will cancel the changed file mode and/or tab size setting.
