BCM_SETLOGFONT

BCM_SETLOGFONT
	wParam = 0;
	lParam = ( LPARAM )( LOGFONT * )lpLogFont;

Set the specified font. The control will create the necessary font from the input LOGFONT structure.

  • Returns non-zero for success and zero for failure
lpLogFont
Pointer to a fully initialized LOGFONT structure describing the font to set. The specified font must be a fixed width font or else this function will fail.
