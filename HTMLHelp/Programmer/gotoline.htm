BCM_GOTOLINE

BCM_GOTOLINE
	wParam = ( WPARAM )( int )nLine;
	lParam = 0;

Moves the caret to the specified line.

  • Returns non-zero for success and zero for failure
nLine
The number of the line to move the caret to. This value must be between 1 and the number of lines in the control contents plus one. Values outside this range will return failure.
