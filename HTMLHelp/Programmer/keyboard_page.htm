This is the Keyboard page of the Brainchild configuration program. With it you can change the keyboard settings of the text class you are editing. The various settings are explaned below:

Mapped keys Here you can see a small toolbar with two buttons,  and . 

With the  button you can add a new key combination and with the  button you delete the currently selected one.
Mapped to Here you see a small toolbar with four buttons, , ,  and . When you press the  button the following small menu appears:

With this menu you can select the type of command you wish to add. Use the image to select the command type for which you want more information.

Selecting the  button from the toolbar will delete the currently selected command. The  and  buttons can be used to move the selected command up or down in the list. The commands are executed in the order in which they are located in the list.
List The list shows you the commands already added to the key combination. Double-clicking an entry will open it's corresponding editor. Either the Hardcoded, Run, Insert or Open command editor.
