BCM_GOTOMARKER

BCM_GOTOMARKER
	wParam = ( WPARAM )( int )nMarker;
	lParam = ( LPARAM )( int )nDirection;

Move the caret to a marker.

    • Returns TRUE for success and FALSE for failure.
nMarker 
The marker to move to. This must be a number between 1 and 5.
nDirection
The direction to move to. This can be any of the following values:

GMK_NEXT - Go to the next line with the specified marker set.
GMK_PREVIOUS - Go to the previous line with the specified marker set.
GMK_FIRST - Go to the first line with the specified marker set.
GMK_LAST - Go to the last line with the specified marker set.
