BCM_NUMCHARSINLINE
	wParam = ( WPARAM )( int )nLineNum;
	lParam = ( LPARAM )( BOOL )bVirtual;

Get the number of character on the line.

  • Returns the size, in bytes, required to hold the line. In case of an error -1 is returned.
nLineNum
The number of the line you want to know the the number of characters in it or -1 in which case the number of characters of the line on which the caret is located will be returned.
bVirtual
TRUE if the number of characters returned should be the number of characters when tabs are converted to spaces.
