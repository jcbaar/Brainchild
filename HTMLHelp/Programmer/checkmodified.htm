BCM_CHECKMODIFIED

BCM_CHECKMODIFIED
	wParam = 0;
	lParam = 0;

Checks to see is the control contents is modified and, if it is, shows a dialog asking the user wether (s)he wan't to save the contents first.

This message is ment to be used before you take action which may result in unsaved changes being lost.

  • Returns zero if the user does not want to continue the following action and non-zero if (s)he does.
