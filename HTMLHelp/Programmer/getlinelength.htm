BCM_GETLINELENGTH

BCM_GETLINELENGTH
	wParam = ( wParam )( int )nLineNum;
	lParam = 0;

Get the size in bytes required to store the line.

  • Returns the size, in bytes, required to hold the line.
nLineNum
The number of the line you want to know the size of or -1 in which case the size of the line on which the caret is located will be returned.
