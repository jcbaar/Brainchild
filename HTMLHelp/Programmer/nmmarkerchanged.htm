NMBC_MARKERSTATECHANGED

NMBC_MARKERSTATECHANGED

	lpnmsc = ( LPNMMARKERSTATE )lParam;

Sent by the control when the state of one of the custom line markers has changed. This notification message is sent in the form of a WM_NOTIFY message.

  • The return value is ignored by the control.
lpnmsc
A pointer to a NMMARKERSTATE structure which contains the information about the line and marker number which has changed.
