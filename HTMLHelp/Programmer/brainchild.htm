Brainchild version 2.5

Documentation

Brainchild is a custom editor control which allows programmers to include powerful and modern editing capabilities to their own applications. The brainchild editor control has many features you would expect from a modern editor.

History

I started brainchild back in 1993 on the Amiga computer. For a couple of years I used it as the editor for all my projects on that machine without ever releasing it to the general public. When the Amiga was all but dead and I switched to the Windows platform I really missed my old editor so I started porting my old code to the Windows OS. Since I used Visual C++ 6.0 I kind of gotten used to the editor this IDE offered so it is no surprise that the current Brainchild version is modelled after the VC++ 6.0 editor.

The first windows running Brainchild was not a custom control. It was a simple SDI application which was a simple port of the Amiga version. Problem was that the old Amiga code was very outdated. It did not have multi-level undo/redo, line length was limited to 1000 bytes and it did not use syntax coloring. In short it was not of this day and age. After having looked at about all the editor source-code on the Web I stumbled accross the CrystalEdit MFC components written by Andrei Stcherbatchenko . These components had two features which were at the top of my list:

  • Multi-level undo/redo.
  • Syntax coloring.

I have used his code to base the undo/redo of the Brainchild control on. I have written my own version of syntax coloring since I wanted to make it customizable for the end user. The CrystalEdit components only understood C/C++ code. With the Brainchild control you can add text classes which have fully configurable syntax-coloring.


Features

  • Based on text classes. Settings can be changed for each class. Classes are defined by the file extention. 
  • Fully customizable syntax coloring.
  • Syntax coloring available both on screen and on printer outputs.
  • Multi-level undo/redo. Maximum number of hops settable.
  • File size only limited by available memory.
  • As many bookmarks possible as there are lines in the text.
  • Fully customizable keyboard.
  • Text selection margin.
  • Line number margin. 
  • OLE Drag and Drop editing capabilities.
  • Automatic and smart indention scemes.
  • etc...
