BCM_INSERTLINE

BCM_INSERTLINE 
	wParam = ( WPARAM )( int )nPosition; 
	lParam = ( LPARAM )( LPCTSTR )pszLine;

Insert a text line at the given position.

  • Returns non-zero for success and zero for failure.
nPosition
The zero-based index position at which the line is inserted or -1 to append the line to the text. When this value is out of range the line is also appended to the text.
pszLine
A pointer to a 0-terminated string containing the line of text to insert.
