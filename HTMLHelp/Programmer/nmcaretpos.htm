NMBC_CARETPOSITION

NMBC_CARETPOSITION
	lpnmcp = ( LPNMCARETPOSITION )lParam;

Sent by the control when the position of the caret changes. This notification message is sent in the form of a WM_NOTIFY message.

  • The return value is ignored by the control.
lpnmcp
A pointer to a NMCARETPOSITION structure which contains the caret position information.
