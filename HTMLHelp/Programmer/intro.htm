Introduction

Have you ever created a program which required text-editing capabilities? I am sure you found that the standard Windows edit control was not the most flexible editing solution you can imagine. There is the Windows RichEdit control but this also does not offer the easiest way to include powerful editing capabilities to your applications.

This does not have to be a problem anymore. With Brainchild you have powerful and modern editing capabilities with only a few lines of code. Features of the Brainchild edit control include:

  • Settings are text class based. The user can create his or her own text classes. The default settings files included with the Brainchild package include some good examples.
  • Control panel based settings program included.
  • Fully customizable syntax coloring for both on screen and printer outputs.
  • Multi-level Undo/Redo. The maximum number of hops is settable.
  • File size only limited by the amount of available memory.
  • Bookmarks. The user can create as many bookmarks as there are lines in the text.
  • Fully customizable keyboard. The user can map any number of commands to a single key combination.
  • Optional selection margin.
  • OLE Drag and Drop editing.
  • Automatic and smart indention scemes.
  • Automatic file backup.
  • etc...
