NMMARGINCLICK

typedef struct tagNMMARGINCLICK
{
	NMHDR	hdr;
	int 	nLine; 
	int 	nMargin; 
	int 	nButton; 
	DWORD 	dwButtonState; 
	BOOL 	bDoDefault; 

} NMMARKERSTATE, FAR *LPNMMARKERSTATE;

Contains information about NMBC_MARGINCLICK notification message.

hdr
NMHDR structure containing information about this notification message.
nLine
The line number on which the marker state is changing/has changed. The line number is 1 based.
nMargin 
The margin in which the click was made. Can contain the following values:

MCT_SELECTION  - the selection margin.
MCT_LINENUMBERS - the line number margin
nButton 
The mouse button which triggered this notification. This can contain any of the following values:
MK_LBUTTON - The left mouse button.
MK_MBUTTON - The middle mouse button.
MK_RBUTTON - The right mouse button.
dwButtonState
Indicates whether various virtual keys are down during the click. This parameter can be any combination of the following values:

MK_CONTROL -  Set if the CTRL key is down.
MK_LBUTTON  - Set if the left mouse button is down.
MK_MBUTTON - Set if the middle mouse button is down.
MK_RBUTTON - Set if the right mouse button is down.
MK_SHIFT - Set if the SHIFT key is down.
bDoDefault
By default this is set to TRUE. The control will check this value uppon return of the notification to see if it needs to perform the default action normally assigned to the click in the margin. If you set this to FALSE before returning the control will not perform the default action.
