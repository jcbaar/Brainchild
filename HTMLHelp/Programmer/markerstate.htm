NMMARKERSTATE

typedef struct tagNMMARKERSTATE
{
	NMHDR	hdr;
	int	nMarker;
	int	nLine;
	BOOL	bState;

	BOOL	bAllowState;
} NMMARKERSTATE, FAR *LPNMMARKERSTATE;

Contains information about NMBC_MARKERSTATECHANGING and NMBC_MARKERSTATECHANGED notification messages.

hdr
NMHDR structure containing information about this notification message.
nMarker 
The number of the custom line marker which is changing/has changed.
nLine
The line number on which the marker state is changing/has changed
bState
For NMBC_MARKERSTATECHANGING notifications this will contain the state to which the marker is about to change. For NMBC_MARKERSTATECHANGED messages this contains the state to which the marker has been changed.
bAllowState
By default this is set to TRUE. The control will check this value uppon return of the NMBC_MARKERSTATECHANGING notification to see if the marker state change is allowed to take place. This value has no significance in the NMBC_MARKERSTATECHANGED notification.
