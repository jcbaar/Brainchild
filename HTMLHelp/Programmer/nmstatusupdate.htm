NMBC_STATUSUPDATE

NMBC_STATUSUPDATE
	lpnmsu = ( LPNMSTATUSUPDATE )lParam;

Sent by the control when the status of the control and or it's contents changes. This notification message is sent in the form of a WM_NOTIFY message.

  • The return value is ignored by the control.
lpnmsu
A pointer to a NMSTATUSUPDATE structure which contains the control and contents status information.
