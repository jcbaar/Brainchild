NMBC_MARKERSTATECHANGING

NMBC_MARKERSTATECHANGING

	lpnmsc = ( LPNMMARKERSTATE )lParam;

Sent by the control when the state of one of the custom line markers is about to change. This notification message is sent in the form of a WM_NOTIFY message.

  • The return value is ignored by the control.
lpnmsc
A pointer to a NMMARKERSTATE structure which contains the information about the line and marker number which is about to change.
