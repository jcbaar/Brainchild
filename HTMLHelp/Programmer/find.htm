This is the find dialog. With it you can search for each occurence of a combination of characters. In the dialog as displayed above you have the following possibilities:

Find what

This combo-box control contains a small history of character strings you have searched for. You can either select one of these strings or type in a new one.

Match whole words only When this option is checked you look for whole words only. This means that when you look for the word "hello" you will find it in text like "hello world" but not in text like "hello-world".
Match case

When this option is checked character casing is also checked.

Find Next When you click on this button the next occurence of the character sequence is looked up.
Find Previous When you click on this button the previous occurence of the character sequence is looked up.
Bookmark All When you click on this button all lines which have a character sequence matching the one you are looking for are bookmarked.
Cancel Aborts the operation.
