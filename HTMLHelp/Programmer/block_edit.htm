This is the block editor dialog. With it you can edit the character sequences that initiate and terminate the text block and it's color.

Block name You can change the name of the text block in here. A name for the block is optional.
This block starts with Type the character sequence that initiates the block in here. This must be valid. An empty block initiator is not allowed.
at the beginning of the line When this option is turned on the block initiator is only recognized when it is the first thing on the line.
when the end of the line is reached Choosing this as block terminator will end the block at the end of the line.
at the first white space This will terminate the block at the first white space or delimiter after the block start.
with the following characters This option will enable the edit control for you to enter the character sequence that terminates the block. If you leave this empty the block will be terminated at the end of the line.
Block color When you click on this button the color selection popup  will open which you can use to change the color used to display this text block.
OK Clicking on this button confirms the changes made.
Cancel Clicking on this button cancels the changes made.
