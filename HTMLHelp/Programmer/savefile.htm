BCM_SAVEFILE

BCM_SAVEFILE
	wParam = 0; // Not used.
	lParam = ( LPARAM )( LPCTSTR )lpszFilename;

Saves the control contents to the specified file.

  • Returns non-zero for success and zero for failure
    • BCMR_OK
      No problems.
    • BCMR_ERROR
      An error occured during the file saving. Use GetLastError() to find out more about the error.
    • BCMR_BACKUP_ERROR
      An error occured during file backup. Use GetLastError() to find out more about the error.
lpszFilename
When this is NULL the current control path is used to save the control contents. Otherwise this should point to a 0-terminated string containing the full path under which the control contents is saved.
